# Agência Criativa Web 🌐✨

Projeto desenvolvido como exercício prático de **criação de um site responsivo para uma agência de design digital**, utilizando conceitos modernos de HTML5 e CSS3.

---

## 📄 Descrição

A **Agência Criativa Web** é um site fictício que apresenta os serviços de uma agência especializada em design gráfico, desenvolvimento web e branding. Este projeto foi desenvolvido com foco em **design responsivo**, adaptável para diferentes tamanhos de tela, utilizando **Flexbox**, **CSS Grid**, **unidades relativas** e **media queries**.

---

## 🚀 Tecnologias Utilizadas

- HTML5 semântico
- CSS3 moderno
  - Flexbox
  - CSS Grid
  - Responsividade com Media Queries
  - Animações e transições
- Git e GitHub para versionamento e hospedagem
- GitHub Pages para publicação do site

---

## 📱 Funcionalidades e Recursos

- Layout 100% responsivo
- Menu de navegação com botão hambúrguer para dispositivos móveis
- Seções organizadas:
  - **Home**: Banner de boas-vindas
  - **Sobre Nós**: Visão da agência
  - **Serviços**: Lista interativa com CSS Grid
  - **Depoimentos**: Comentários de clientes fictícios
  - **Contato**: Formulário com foco e destaque visual
- Animações suaves e interações modernas
- Acessibilidade básica aplicada
